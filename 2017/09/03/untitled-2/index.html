Matrix multiplication : The inner product of matrix is an alternative to For loop: it concerns the process of element wise multiplication and addition of two one-dimentianal matrices. If the involved matrix is 2-dimentional, the process produces a stacked matrix that stores matrix-wise point multiplication results.

Computation graph :导数的链式连接方式： 代价函数在当前变量的偏微分等于代价函数在后一变量（也即当前函数输出变量）的偏微分 * 当前函数对当前变量的偏微分。 dL/db = dL/da * da/db
